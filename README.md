# Statistics in Python Project One

## Source: 
- 'https://www.kaggle.com/datasets/mysarahmadbhat/lung-cancer'

Public Data set based on a survey consisting of 309 participants.
The dataset contains questions related to lung cancer and related risk factors.
Including:
- Gender
- Age
- Smoking habits
- Anxiety levels
- Chronic disease presence
- Fatigue levels
- Allergy presence
- Wheezing presence
- Alcohol consumption
- Coughing presence
- Shortness of breath
- Swallowing difficulty
- Chest pain presence

The target variable is whether the participant has lung cancer or not.

## Displaying Information:
- Box plots for Gender
- Scatter plot for Age
- Box plots for Smoking habits
- Box plots for Alcohol consumption
- Box plots for Allergy presence
